This repository contains the HTML, CSS, and JavaScript source code for my interactive portfolio. It showcases my professional capabilities, core competencies, and featured case studies in Data Analytics, Machine Learning, Data Engineering, and Business Strategy.

## 🛠️ Tech Stack
This website was built using a modern, lightweight, vanilla tech stack to ensure maximum performance and responsiveness:
* **HTML5:** Semantic structure and accessibility.
* **CSS3:** Custom styling, glassmorphism UI, CSS variables, and responsive flexbox/grid layouts.
* **JavaScript:** Dynamic modal injection and interactive UI components.
